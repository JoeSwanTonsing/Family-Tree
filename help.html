Getting Started

To get started, click on the "Get Started" button at the home page, or the "Create" link in the navgation bar


OR

Menus

There are three Menus


1. Use the Menu numbered 1 to open up options to export your tree. Currently, you can export tree as "PDF" or as a "PNG" file. More export options will be added.

2. Use the Menu numbered 2 to open up options for a particular tree node - member(Parent/Child). Use this menu to Edit, Add or Remove a member.

3. Menu numbered 3 is a context menu and is accessible only through right-click button of the mouse and may not be accessible to all devices. The context menu consists of the same menu items as that in menu 2.

Details

You can easily view or edit the details of a member

To view the details of a member, click/tap anywhere on the member's card. This will bring up the details of the member.

To edit the details of a member, click on the menu () within the member's card and select the "Edit" option. This will bring up the details of the member in an editable form.
As of now, user can only edit the Name attribute. All other fields will be in working condition soon.

Finally, to save the changes, click on the arrow () at the top right corner. Your member node should update automatically.

Add Member

To add a member of the family, use the "Add Child" option from the node menu.

If member is the first generation, on the node of "Family Name" add the first member. For any other generation, adding a child node automatically makes the new child node as the child of that node.


Adding Spouse

Since a spouse is not a direct child of the parent, you can't add it as a child node from the parent node. To add a spouse, goto the node whose spouse you want to add. Click on the node options and select the "Add Child" option. Then on the generated child node, "Edit" the child node - Enter the details and on the bottom, Select the "Spouse" option. This option differentiates the Spose node from the ordinary child nodes.

Navigating

When the tree gets too big, it doesnt fit in the whole screen. To navigate through the tree, click and hold on any empty area of the tree (outside the nodes) and drag in ay direction.
